# Realm of 5 Crowns — Unity Scaffold

Use these starter scripts inside a Unity project for the Obsidian Gate prototype.

## Suggested structure

```text
unity/
  Scenes/
    ObsidianGate.unity   # Create in the Unity editor
  Scripts/
    PlayerController.cs
    EncounterTrigger.cs
    GateWardenEncounter.cs
```

The scene asset itself should be created in the Unity editor, then wired to the scripts in `Scripts/`.
