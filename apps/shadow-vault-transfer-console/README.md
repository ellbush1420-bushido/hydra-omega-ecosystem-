### B. Jezebel Velvet Character Forge
Purpose: Character and persona development for cinematic production.

Functions:

- Faction assignment
- Persona metadata
- Mood, tone, and visual identity fields
- Prompt lineage and continuity notes

Use cases:
- Nyssa recruitment scenes
- Velrya covert-arc reels

- Den Mother / Spider Seminary teaser sequences

---

### C. Velvet Python Motion Engine
Purpose: Convert static character concepts into structured motion-ready reel briefs.

Functions:
- Face-to-character motion test planning
- 5–15 second faction recruitment clips
- Motion reference briefs
- Character continuity checkpoints
- Movement beat sequencing

Output example:
1. Hook image / pose
2. Motion transformation cue
3. Voice or sound moment
4. Faction reveal
5. CTA ending

---

### D. Shadow Moon Cinematic Reel Builder
Purpose: Assemble short-form lore clips for social deployment.

Functions:
- 5–15 second reel blueprint
- Hook → motion → lore → CTA layout
- Platform tag: TikTok / Instagram Reels / Shorts
- Export checklist
- Caption and overlay planning

Campaign 001 preset:
- “Choose Your Crown” clip generator
- Recruitment CTA variants
- Crown/faction identity tag
- Product or Discord call-to-action

---

### E. Productization Studio
Purpose: Convert repeatable creative workflows into monetizable products.

Product classes:
1. **Character Motion Prompt Packs**
2. **15-Second Lore Reel Templates**
3. **Hydra Cinematic Hook Kits**

Suggested fields:
- Product title
- Product tier
- Included assets
- Prompt pack count
- Reel template count
- Hook count
- Price point
- Vault status
- Release status

---

### F. Shadow Vault Transfer Queue
Purpose: Govern what moves from raw experiment to approved, productized, or community-released asset.

Transfer statuses:
1. Draft
2. In Review
3. Approved
4. Vaulted
5. Productized
6. Conclave Released
7. Archived

Required metadata:
- Character
- Faction
- Campaign
- Asset type
- Prompt lineage
- Rights / safety status
- Reviewer notes
- Release eligibility
